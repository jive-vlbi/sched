AZCOLIM

AZCOLIM sets the azimuth colimation offset to add to the nominal one used by the VLBA on-line computers for pointing. Usually this will be 0.0. This allows the pointing for a VLBA antenna to be adjusted.

  • Argument: A pointing offset in arc minutes.
  • Options: Any real number.
  • Default: 0.0
  • Usage: Defaults to previous station.
  • Example: AZCOLIM=1.2
