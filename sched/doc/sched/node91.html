FASTFOR

FASTFOR: requests that the wide band tape (Mark III or VLBA) be run at the maximum foreward speed during the prescan. Usually the intent is to get to the end of the tape prior to the start of the scan. If no argument is given, all stations will be issued with a fast forward instruction. If a list of stations is given, only those stations in the list will be fast forwarded. The station code may be used instead of the station name (not case sensitive).

  • Argument: None or a list of stations.
  • Options: None or a list of any stations in the scan. Not case sensitive.
  • Default: No fast forward unless there is insufficient room on the tape to complete the scan while recording in the current direction.
  • Usage: Reverts to no fast forward on next scan.
  • Example: FASTFOR or FASTFOR=VLBA_BR
