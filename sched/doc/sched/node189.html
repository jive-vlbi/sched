VLAMODE

VLAMODE gives the VLA observing mode. The valid options are:

' ' - normal VLA continuum

'VS' - single antenna VLBI

'VA' - auto phasing on VLA A (RCP) and VLA D (LCP) IFs

'VB' - auto phasing on VLA B (RCP) and VLA C (LCP) IFs

'VR' - auto phasing on VLA A and VLA B (both RCP) IFs

'VL' - auto phasing on VLA C and VLA D (both LCP) IFs

'VX' - extended phasing (holding phases from previous VA, VR or VL scan.)

'IR' - measure pointing offsets. See VLAPEAK for details.

Other valid options are 'IA', 'IB', 'IC', 'PA', 'PB', 'PC', 'PD', 'D ', 'TE', 'TF', 'O ', and 'ON'. Consult VLA documentation for details.

  • Argument: Text of 2 characters.
  • Options: See list above.
  • Default: None - required for VLA.
  • Usage: Defaults to previous scan.
  • Example: VLAMODE='VX'
