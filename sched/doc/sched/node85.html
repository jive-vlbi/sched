ELCOLIM

ELCOLIM can be used to specify an increment to the elevation colimation offset value used for pointing. This is added to the nominal position known by the on-line system for the particular receiver and to the value specified in the setup file. It is also added to the values used for offsets in pointing patterns and to any value from the setup file.

  • Argument: A elevation colimation offset in arc minutes.
  • Options: Any value.
  • Default: 0.0
  • Usage: Default to previous scan.
  • Example: ELCOLIM = 0.25
