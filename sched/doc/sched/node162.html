SATFILE

SATFILE is used to specify the name of a file of orbital elements for satellite observations. This is normally only used for single dish testing, specifically holography.

  • Argument: A character string of up to 80 characters. Should be a file name.
  • Options: Any character string.
  • Default: 'NONE' -- means don't open such a file.
  • Usage: Only one used per experiment.
  • Example: SATFILE = 'orbits.2000jan15.dat'
