Version 6.05

Version 6.05 is a minor revision from version 6.04. The main issue addressed was a problem where sources between 0 and -1 deg dec had the wrong sign of dec in sources.vlba. There are also a couple of bug fixes in the plot routines and new Arecibo entries in freq.dat.

  • Correct sign of declination for many sources in the range 0 to -1 degrees. June 2, 2006 RCW.

  • Complete update of Arecibo frequency data from Chris Salter June 5, 2006.

  • Fix plot programs to deal with multiple setups properly and to fix a recent bug in the setting of plot scales. Franco Tinarelli, June 7, 2006.

  • Reflect offending line when there is a syntax crror. June 8, 2006 RCW.
