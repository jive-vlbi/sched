CORNANT

CORNANT informs correlator operations of the number of stations scheduled. This helps determine when all logs and tapes are in and helps determine what the correlator load, both in terms of drives needed and output data rate, will be. See the discussion of CORCHAN for a formula to calculate the output data rate.

  • Argument: An integer number.
  • Options: Any, but it should be the expected number of antennas.
  • Default: Number of antennas in schedule.
  • Usage: Only one value used -- the last.
  • Example: CORNANT = 10
