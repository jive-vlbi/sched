VLAIF

VLAIF gives the name of the VLA IF file that contains gain calibration and efficiency information.

Note that, if any of FLUKEA, FLUKEB, VLAFEAB, VLAFECD, VLASYNA, VLASYNB, VLAIF, FEFILTER, or VLAROT are specified, both LO and FI cards will be made. In such cases, any other parameters allowed to default will be set to the standard value for the specified VLABAND.

  • Argument: Up to 10 character file name.
  • Options: Any string.
  • Default: Uses VLA default for the specified band.
  • Usage: Defaults to previous station.
  • Example: VLAIF = 'SYSLIF'
