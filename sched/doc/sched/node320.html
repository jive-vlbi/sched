Release of 16 May 1995

  • TAVLBA mode, make second on-source qual 20 and add final off. (1994 December 21)

  • Bug fix: if one velocity was given in the source catalog, it was not used for other baseband channels. (1994 December 30)
  • Add tape summary to summary file. (1995 January 24)

  • Add setup name to summary listing. (1995 January 24)

  • Add third pcal tone when there are enough detectors. (1995 January 25)

  • Fix source name in error message when there are imbedded blanks. (1995 April 24)

  • Add to summary file a list of sources in format for transfer to the correlator data base. (1995 May 15)

  • Turn on UNLOAD between tapes at 1-drive stations. (1995 May 16)

  • Fix problem with tape summary for stations excluded by DOSTA. (1995 May 16)

  • Don't print the ``.set'' in the summary file if the setup file name is too long to fit. (1995 May 16)

  • Adding commands to control the 50cm filter.

  • Improved the reliability of writing the EXTLO parameter. (1995 May 16)

Consult the old sched.hlp file for changes before 1995. It can be obtained from Craig Walker.
