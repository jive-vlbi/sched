VLABAND

VLABAND is the band code for VLA projects, whether VLBI or VLA-only. For complete information on the standard bands, see the VLA Standard Bands section.

Here is a brief list bands not including those involving 75 MHz. One of the standard bands must be listed even if the frequency will be modified with other control parameters. This helps insure that all VLA setup files etc are correctly specified.

SCHED can determine the value of this parameter from the frequency catalog if that file includes an entry that covers the desired VLA setup.

  • Argument: Text of 2 characters.
  • Options: 'VP' - 90cm VLBI. 'VL' - 18cm VLBI. 'VC' - 6cm VLBI. 'VX' - 4cm VLBI. 'VU' - 2cm VLBI. 'VK' - 1cm VLBI. 'VQ' - 7mm VLBI. 'PP' - 90cm VLA-only. 'LL', 'HH', '18' - ``20cm'' VLA-only. 'LP' - Mixed 20cm and 90cm. 'CC' - 6cm VLA-only. 'XX' - 4cm VLA-only. 'UU' - 2cm VLA-only. 'KK' - 1cm VLA-only. 'QQ' - 7mm VLA-only.
  • Default: None - required for VLA. Can be specified in the frequency catalog.
  • Usage: Defaults to previous scan.
  • Example: VLABAND='VC'
