COMMENT

COMMENT is a string of up to 64 characters to be written on operator and telescope files just before the scan.

  • Argument: Text up to 64 characters.
  • Options: Any.
  • Default: Blank
  • Usage: Reverts to no comment if not specified for each scan.
  • Example: COMMENT='ET, call home.'
