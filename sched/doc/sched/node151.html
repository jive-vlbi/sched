PTLINK

Specifying PTLINK tells SCHED  that these are observations involving the Pie Town link. SCHED  will write //LO lines in the ...obs.y control file for the VLA and the Pie Town first LO frequency will be given on those lines. Probably other functions will be added as this capability comes on-line.

  • Argument: None -
  • Options: None
  • Default: Not set.
  • Usage: Only one value accepted, the last.
  • Example: PTLINK
