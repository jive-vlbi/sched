FLUKESET

FLUKESET specifies which set of Fluke synthesizers to use at the VLA. Usually this should be 1 for phased array or normal VLA observations and 2 for single antenna VLBI. The VLA operators should be warned to check this in case the normal pattern is not being followed. The default 0, written as blank, may usually be best. Let the VLA operator select the flukeset.

  • Argument: Integer.
  • Options: 1 or 2.
  • Default: 0 - written as blank.
  • Usage: Defaults to previous station.
  • Example: FLUKESET=1
