Release of 14 Jan. 1997

  • Fixed mistake in frequencies in v6cm-128-4-2-L.set. Put in 4 Dec release. 5Dec96

  • Add source qualifier to qualifiers used in pointing pattern. This will be used to distinguish scans meant to be processed with on-off spectral line observations. For testers only. 6 Dec 96 RCW.

  • Fix bug in SLEW that caused slew times to be wrong some of the time. This was introduced in the 4 Dec sched so old schedules are ok. 9 Dec 96 RCW.

  • Rick Lively has been doing some clean up in the Makefile, especially for machines other than those we have. The 9 Dec release has several modifications relative to the 4 Dec release.

  • I did a mini-release in the evening of 9-Dec-1996 RCW

  • Franco Tinarelli sent information on a number of problems detected by his Dec Alpha compiler. Mostly these are non-serious problems with declares (although he is the one who flagged the SLEW problem above). These are fixed. 10 Dec 1996. RCW

  • Block test for different speedups for S2 sites. 13 Dec 96 RCW

  • Make RCHAN and LCHAN have 2 characters. 13 Dec 96 RCW

  • Add setup information to the end of the sch files. 13 Dec 96 This and the above 2 items were requests from Huib.

  • Alter TFORM so that it does not overflow a small input character string. RCW 18dec96.

  • Start adding SAVE to routines and includes. Needed with g77. RCW 18dec96. First change in KEYPUT. 23dec96 -- this turned into a mess. Putting variable specific SAVEs in many routines. In the process, a few initializations were cleaned up.

  • Reinstate postpasses for VLBA format files. 23 dec 96. RCW.

  • Write verify and check_results scripts for checking both new code and new installations. Put ``standard'' output files in the examples/Stdout subdirectory. 26 Dec 96. RCW.

  • Do various programming cleanups to keep g77 with all warning switches turned on happy. These should make the program more robust.

  • Fix several problems for RESTARTs from the plotting section. The fixes mainly had to do with the initialization of station and frequency catalog reads. 31 Dec 96.

  • Allow single tape stations not to postpass if the tape doesn't stop during a pass. 31 Dec 96.

  • Warn if tape change gaps are less than 15 minutes. 13 Jan 97 RCW.

  • If a tape change is preceeded by norecord scans, move the tape change time back to as early as possible. 13 Jan 97 RCW.
