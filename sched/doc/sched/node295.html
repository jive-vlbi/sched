Release of 10 Feb. 1997

  • Fix a concatenation problem (Linux only?) in envir.f - related to the use of environment variables. RCW 15 Jan 1997. Moved to standard release (14 Jan version).

  • Add some variables to SAVEs. Required by g77 to fix some problems. 21 Jan 1997.

  • Change the order of the examples to try to get people to use the simpler one. 21 Jan 1997.

  • Update Sla/dat.f for July 1, 1997 leap second. 27jan97 RCW.

  • Add comment to crd files giving tape start times. 3feb97 RCW.

  • Incorporate VEX output from Huib. Debugging and porting over next few days. 3feb97 RCW.

  • Prevent mixing planets and VEX. 6feb97 RCW.

  • Fix MST day for VLA obs files when the experiment starts between UT and local midnight. 7Feb97 RCW.

  • This should be the most robust release yet in a portability sense because of testing on several platforms, much of it done by Franco Tinarelli of Bologna. Numerous small non-standard items in the code were corrected.
