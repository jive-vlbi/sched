Release of 18 August 1997.

  • Calls to jpleph for planet positions wanted East Longitude. They had been getting West Longitude. Fixed 5 Aug 1997. RCW.

  • Add PRECDATE - the "observation" date to assume for precession from J2000 to B1950 etc. 5 Aug 1997. RCW.

  • Stop using CHNSTA in the frequency table and comment out the code that implements it. Because of the digital patch panel at the VLA and the effort by sched to understand which IFs are needed to support the specified phasing modes (VL, VR, and VA), CHNSTA is no longer needed. 8Aug97. RCW

  • Add barrel=roll_off to the vsop setup files. 8Aug97

  • Install better documented versions of the planets software from Brian Butler. 8Aug1997.

  • Restructure setup file expansion in SETEXPND to avoid bug when the number of setup groups shrinks. 15Aug1997. RCW.

  • Protect against requesting VLBI in non-VLBI modes at the VLA. This would create uncorrelatable data. 15Aug1997. RCW.

  • Fixed bug that lead to negative pcal frequencies in certain narrow bandwidth cases. 18Aug1997. RCW.

  • Fixed keeping track of use of VLA modes. Before was at time of input, but got setup group index wrong. Now in setup defaulting routines. 18Aug1997. RCW.
