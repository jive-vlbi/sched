ELCOLIM

ELCOLIM specifies the elevation colimation offset to be added to the one used by the VLBA on-line computers. It can be used to adjust the VLBA pointing.

  • Argument: A pointing offset in arc minutes.
  • Options: Any real number.
  • Default: 0.0
  • Usage: Defaults to previous station.
  • Example: ELCOLIM=1.2
