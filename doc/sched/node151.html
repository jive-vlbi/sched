PEAKFILE

PEAKFILE is used to specify an external file containing parmeters to control automatic insertion of reference pointing scans. The default is the file provided with the SCHED  distribution. The same information can be put in the main SCHED  input using PEAKINIT.

  • Argument: A file name of up to 80 characters
  • Options: Any file name
  • Default: $SCHED/catalogs/peak.cmd on unix systems.
  • Usage: Only the last value specified is used.
  • Example: peakfile=mypeak.cmd
