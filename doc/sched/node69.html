CORTAPE

CORTAPE specifies the type of tape that should be used for the data distribution from the correlator. This must be DAT, EXABYTE, FTP or NONE. The latter two basically mean that the PI will pick up a disk copy of the data by means of a network copy (there is no effective difference between the two).

  • Argument: A string of up to 16 characters
  • Options: DAT, EXABYTE, or FTP
  • Default: Blank -- Not too useful.
  • Usage: Only one used -- the last.
  • Example: CORTAPE = 'DAT'
