DODOWN

Normally, SCHED  eliminates stations for which the source is not up if it is a VLBA station with AUTOTAPE set to 2 or if it is any station using a disk recording system. DODOWN with no argument overrides that behavior and keeps stations in scans regardless.

  • Argument: None
  • Options: No argument
  • Default: Stations taken out of scans (DODOWN not zero)
  • Usage: Only one value, the last, used for schedule.
  • Example: DODOWN
