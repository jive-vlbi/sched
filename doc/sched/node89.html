EXPCODE

EXPCODE is the project code from the Network or from NRAO. This is used as the first few characters of many file names. Only the first 6 characters are used in the VLA observe files.

  • Argument: Text of up to 8 characters. Best to use 6 or less. In fact, only 5 characters will be kept by much of the NRAO bookkeeping system.
  • Options: Any.
  • Default: 'NUG'
  • Usage: Only one value used, the last given.
  • Example: EXPCODE=BW005
