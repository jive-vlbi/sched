FLUKEA and FLUKEB

FLUKEA and FLUKEB are used to set the VLA fluke synthesizer frequencies. See the discussion of FIRSTLO for more details on what this does to the observing frequency. The numbers to give are the numbers for the ``//FI'' card in the VLA observe file, which is twice the actual synthesizer setting for FLUKEB but is the frequency mixed with the data. To get the same frequency on the VLA's AC and BD IFs, make FLUKEB equal FLUKEA plus 100.00.

Note that, if any of FLUKEA, FLUKEB, VLAFEAB, VLAFECD, VLASYNA, VLASYNB, VLAIF, FEFILTER, or VLAROT are specified, both LO and FI cards will be made. In such cases, any other parameters allowed to default will be set to the standard value for the specified VLABAND.

FLUKEA and FLUKEB can be provided in the frequency catalog.

  • Argument: A real number giving the frequency in MHz.
  • Options: FLUKEA: Values between about 100 and 150. FLUKEB: Values between about 200 and 250.
  • Default: The VLA default for the band is used.
  • Usage: Defaults to previous station.
  • Example: FLUKEA=102.11 FLUKEB=202.11
