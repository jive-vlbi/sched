AZCOLIM

AZCOLIM can be used to specify an increment to the azimuth colimation offset value used for pointing. This is added to the nominal position known by the on-line system for the particular receiver and to the value specified in the setup file. It is also added to the values used for offsets in pointing patterns and the value from the setup file.

  • Argument: An azimuth colimation offset in arc minutes.
  • Options: Any value.
  • Default: 0.0
  • Usage: Default to previous scan.
  • Example: AZCOLIM = 0.25
