ADDRESS1, ADDRESS2, ADDRESS3 and ADDRESS4

ADDRESS1, ADDRESS2, ADDRESS3, and ADDRESS4 are used to give the Principal Investigator's address for the cover information. At least the first line must be provided if recording tape.

  • Argument: A character string of up to 64 characters for each of the 4 parameters.
  • Options: Any.
  • Default: Blank
  • Usage: Only one value used, the last.
  • Example: ADDRESS1='1003 Lopezville Rd'
    ADDRESS2='Socorro, NM 87801
    ADDRESS3=' U S A '
